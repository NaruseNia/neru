# neru 実装計画書

## 概要

本書は neru スクリプト言語の実装計画を定義する。
全5フェーズで段階的に開発を進め、各フェーズの完了時点で動作可能な成果物を得る。

- **実装言語**: Zig
- **ビルドシステム**: Zig Build System (`build.zig`)
- **テスト**: Zig 組み込みテストフレームワーク

---

## 補足: 非機能要件

### パフォーマンス目標

- コンパイル: 10,000 行のスクリプトを 1 秒以内でコンパイル
- VM 実行: 演出命令のディスパッチが 1ms 以内
- メモリ: VM のベースメモリ使用量 10MB 以内

### テスト方針

- 各モジュールに単体テスト
- Phase ごとに統合テスト用の .neru サンプルスクリプト
- エッジケース（空ファイル、巨大ファイル、不正構文、深いネスト等）のテスト

### ドキュメント

- `docs/specification/` に仕様書を維持
- README.md にクイックスタートガイド
- サンプルプロジェクト（`examples/`）の提供
